# Agent Publisher 产品演进路线图

> 最后更新：2026-04-04

## 产品定位

**AI 驱动的内容经营工作台**——从热点发现、AI 创作、精修编辑到多平台分发的一站式内容生产系统，覆盖图文、演示稿和视频三种内容形态。

## 核心价值主张

| 环节 | 价值 |
|------|------|
| 发现 | 全网热榜聚合，不再从空白页开始 |
| 创作 | AI + 模板 + 风格预设，5分钟出稿 |
| 编辑 | 富文本 + 微信预览，所见即所得 |
| 分发 | 一键推送草稿箱，多账号批量发布 |
| 变现 | Credits 计费体系，按用量付费 |

## 演进阶段

### 阶段一：核心路径打通（v1.0 → v1.5）

**目标**：新用户 5 分钟内从热点生成一篇文章并预览

#### 1.1 内置默认 Agent + 零配置启动
- [ ] 系统启动时自动创建内置 Agent "锐评官"（风格鲜明的科技观察视角）
- [ ] 用户无需手动配置 Agent 即可开始创作
- [ ] 内置 Agent 提供高质量系统提示词，引导 AI 输出有辨识度的内容风格
- [ ] 用户随时可修改内置 Agent 或创建自己的 Agent

#### 1.2 wenyan-cli 自动安装 + 排版保障
- [ ] 后端启动时检测 `wenyan` CLI 是否存在
- [ ] 未安装时自动执行 `npm install -g @aspect-build/wenyan` 或同等方式
- [ ] 确保所有文章 Markdown → HTML 渲染走 wenyan，排版样式一致
- [ ] 开发文档中明确 wenyan 依赖和安装说明

#### 1.3 创作控制台与文章编辑器融合
- [ ] 将 Create.vue 和 ArticleEditorPage.vue 合并为统一的创作控制台
- [ ] 三列布局（左配置 / 中编辑 / 右预览）同时支持"AI 生成"和"手动编辑"
- [ ] 文章管理列表点击"编辑"→ 直接在创作控制台打开（`/create?article_id=xxx`）
- [ ] 用户在创作时可输入自定义提示词，系统自动融合 Agent 系统提示词 + 风格预设 + 用户提示词

#### 1.4 用户提示词 + 系统提示词融合创作
- [ ] 创作弹窗 / AI 操作栏增加"创作指令"输入框
- [ ] 用户输入的提示词与 Agent 的系统提示词、风格预设、提示词模板组合后发送给 LLM
- [ ] 支持三种创作模式：爆款二创 / 热点总结 / 观点扩写
- [ ] 拼接顺序：系统提示词（Agent）→ 风格预设 → 提示词模板 → 用户自定义指令

#### 1.5 信息架构重构
- [ ] 22 个页面收敛为 4 个核心页面 + 设置区
- [ ] 主导航：发现 → 创作 → 内容 → 数据
- [ ] 设置区：公众号、Agent、数据源、提示词、LLM、Credits

#### 1.6 发现→创作一键路径
- [ ] 热点卡片增加"一键创作"按钮
- [ ] 快速创作弹窗（选 Agent、创作模式、创作指令）
- [ ] 生成后直接进入创作控制台，素材对照 + 微信预览

#### 1.7 消灭产品代码中的 Mock 数据
- [ ] 趋势曲线：存储每次采集的 quality_score 快照，展示真实趋势
- [ ] SearchCollector：替换 StubSearchCollector，接入真实搜索能力
- [ ] 保留支付占位（Credits 系统在阶段二落地）

#### 1.8 Credits 计费体系设计
- [ ] 统一 Credits 模型（生文、生图、生视频不同消耗）
- [ ] 用户 Credits 余额管理
- [ ] Credits 消耗记录与审计
- [ ] 套餐包定义（免费额度 + 付费套餐）

### 阶段二：体验打磨 + Credits 落地（v1.5 → v2.0）

**目标**：完成商业化闭环，用户愿意为 Credits 付费

#### 2.1 热点体验升级
- [ ] 展示阅读量/互动量（从 hot_value 提取）
- [ ] 跨平台标签（"微博+知乎+头条 三平台上榜"）
- [ ] 时间衰减标签（"2小时前上榜"、"持续热议6小时"）
- [ ] 真实趋势曲线（每30分钟采集快照）

#### 2.2 编辑器增强
- [ ] 样式模板一键切换
- [ ] 图片流程优化（COS → CDN → 微信兼容）
- [ ] 段落级 AI 改写（选中 → "换个说法"）

#### 2.3 Credits 系统上线
- [ ] 前端 Credits 仪表盘
- [ ] 生成前 Credits 余额校验
- [ ] 免费用户每日基础额度
- [ ] 付费套餐（占位支付 → 人工激活）

#### 2.4 转化组件
- [ ] Landing Page（价值展示 + 注册引导）
- [ ] Credits 用量进度条
- [ ] 新用户 7 天试用期

### 阶段三：多形态内容 + 自动化（v2.0 → v3.0）

**目标**：从图文扩展到演示稿和短视频

#### 3.1 演示稿生成
- [ ] 文章 → Slideshow 章节拆分
- [ ] timeline.json 编排
- [ ] 竖屏/横屏模板

#### 3.2 视频化输出（Remotion）
- [ ] TTS 语音合成（notes → 音轨）
- [ ] Remotion 渲染服务集成
- [ ] 视频 Credits 消耗定价
- [ ] 视频预览与导出

#### 3.3 高级自动化
- [ ] 定时热点采集 + 自动选题推荐
- [ ] Agent 自动巡航模式（采集→筛选→生成→待审）
- [ ] 发布日历视图

## Credits 体系设计

### 消耗标准

| 操作 | 消耗 Credits | 说明 |
|------|-------------|------|
| AI 生成文章 | 10 | 一次完整的文章生成（含标题+摘要+正文） |
| AI 段落改写 | 2 | 选中段落 AI 重写 |
| AI 扩写/缩写 | 3 | 基于素材的扩写或压缩 |
| 混元生图 | 5 | 一张 AI 配图 |
| 视频生成 | 20 | 一个短视频渲染（后期） |
| 热点导出 CSV | 1 | 批量导出热点数据 |

## 竞品参考

详见 [竞品分析文档](./competitor-analysis.md)

核心差异化：
- **baolem**：闭环短、商业化成熟，但底层能力弱
- **Agent Publisher**：底层能力强（多模型、多账号、调度），需要补产品闭环

## 技术依赖

- **热点采集**：NewsNow API（灵感来源 TrendRadar）
- **AI 写作**：Claude / 多模型 LLM 适配
- **AI 配图**：腾讯混元文生图
- **视频渲染**：Remotion（规划中）
- **云存储**：腾讯云 COS + CDN
