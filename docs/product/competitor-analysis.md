> 从 `竞品/baolem-竞品分析.md` 迁移并补充，最后更新：2026-04-03

## 竞品概览

### baolem.com

**定位**：AI 公众号内容生产 SaaS

**核心页面**：
1. **每日爆款** — 热点内容聚合 + 生产入口
2. **提示词库** — 可复用 Prompt 资产
3. **智能生文** — 文章生产工作台
4. **一键排版** — 发布前样式编辑
5. **一键二创** — 爆款内容快速复刻
6. **会员中心** — 订阅制商业化

**盈利模式**：`AI 内容 SaaS + 订阅制 + 配额限制`

### 对比矩阵

| 维度 | Agent Publisher | baolem.com | 差距 |
|------|----------------|------------|------|
| **底层 LLM** | 多模型适配（Claude/混元/自定义） | 单模型 | 我们强 |
| **热点采集** | 11 平台真实 API + 权重算法 | 有采集能力 | 相当 |
| **多账号管理** | Agent 机制 + 多公众号 | 分组管理 | 我们强 |
| **产品闭环** | 弱（页面分散、路径长） | 强（一键二创） | 差距大 |
| **商业化** | 占位（无在线支付） | 成熟（月/季/年卡） | 差距大 |
| **排版体验** | TipTap 基础 | 模板+配色完整 | 差距中 |
| **热点→创作绑定** | 需跳转多页 | 弹窗一键二创 | 差距大 |
| **定时调度** | APScheduler 完整 | 未见 | 我们强 |
| **视频化能力** | Slideshow + Remotion 规划 | 无 | 我们强 |
| **配图生成** | 混元文生图 | 未见 | 我们强 |

### baolem 做得好的地方

1. **闭环短**：看到爆款 → 一键二创 → 排版 → 草稿箱，步骤最少
2. **爆款页 = 生产入口**：每条内容旁直接挂动作按钮
3. **商业化包装成熟**：试用、折扣、套餐、社证完整
4. **提示词资产化**：不是一次性，而是可复用
5. **排版最后一公里完整**：样式、颜色、字体、封面

### 我们的差异化优势

1. **多模型 LLM**：不绑定单一模型，可选 Claude/Haiku/自部署
2. **Agent 机制**：每个号独立配置主题、风格、数据源
3. **视频化路线**：文章 → 演示稿 → 视频的完整规划
4. **AI 配图**：混元文生图集成
5. **自动化调度**：定时采集 + 批量生成 + 自动推送
6. **开源可控**：完整代码掌控，可自部署

## 竞争策略

**不正面对标 baolem 的闭环体验，而是在底层能力的基础上，走差异化路线**：

1. **短期**：补齐核心闭环（发现→创作一键路径），达到 baolem 的基础体验
2. **中期**：Credits 体系上线，以"按用量付费"区别于 baolem 的"按时间订阅"
3. **长期**：视频化能力落地，拉开产品形态差距
