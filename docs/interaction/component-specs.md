# 关键组件交互规格

> 最后更新：2026-04-04

## 1. 热点卡片 (HotspotCard)

### 状态

| 状态 | 视觉 | 触发条件 |
|------|------|---------|
| 默认 | 白色卡片，底部边框 | 初始状态 |
| 悬停 | 上浮2px + 阴影 + 按钮显现 | mouse hover |
| 已收藏 | 收藏图标高亮 | 用户已收藏 |
| 已创作 | 右上角"已创作"标签 | 已从该热点生成文章 |

### 数据字段

```
┌─────────────────────────────────────────────┐
│ {platform_icon} #{rank} {platform_name}      │
│ · {time_label}                              │
│                                             │
│ {title}                                     │
│ {summary}  (折叠状态仅显示1行)               │
│                                             │
│ {heat_bar} 热度 {quality_score}             │
│ {cross_platform_tags}                       │
│                                             │
│        [📝 一键创作]   [📋 收藏素材]          │
└─────────────────────────────────────────────┘
```

### time_label 规则

| 条件 | 显示 |
|------|------|
| < 1小时 | "刚刚上榜" |
| 1-6小时 | "X小时前上榜" |
| 6-24小时 | "持续热议X小时" |
| > 24小时 | "X天前" |

### heat_bar 规则

- 0.0-0.3: 低温（灰色）
- 0.3-0.6: 升温（黄色）
- 0.6-0.8: 高热（橙色）
- 0.8-1.0: 爆热（红色）

## 2. 快速创作弹窗 (QuickCreateDialog)

### Props

```typescript
interface QuickCreateDialogProps {
  visible: boolean;
  hotspot: {
    id: number;
    title: string;
    summary: string;
    original_url: string;
    platform: string;
    quality_score: number;
  };
}
```

### 表单字段

| 字段 | 类型 | 必填 | 默认值 |
|------|------|------|--------|
| agent_id | Select | 是 | 用户唯一Agent或上次使用 |
| mode | Radio | 是 | "rewrite" |
| extra_prompt | Textarea | 否 | 空 |

### 创作模式选项

| 值 | 显示 | 说明 |
|----|------|------|
| rewrite | 爆款二创 | 基于原文二次创作 |
| summary | 热点总结 | 多角度综合总结 |
| expand | 观点扩写 | 展开个人观点分析 |

### 提交流程

```
点击"开始生成"
  → disabled 按钮，显示 loading
  → POST /api/credits/check { cost: 10 }
  → 如果余额不足，显示余额不足弹窗
  → 如果充足，关闭弹窗
  → router.push('/create?hotspot={id}&agent={agent_id}&mode={mode}')
```

## 3. Credits 指示器 (CreditsIndicator)

### 位置

顶部导航栏右侧，头像左边

### 样式

```
正常:    [💎 450]  蓝色文字
低余额:  [💎 15]   橙色文字 + 轻微脉冲动画
耗尽:    [💎 0]    红色文字
```

### 低余额阈值

- 套餐 Credits 的 10% 以下触发橙色警告
- 免费用户：< 10 Credits
- 基础版：< 50 Credits

### 点击交互

点击展开下拉面板，显示：
- 余额明细（免费 / 付费）
- 本月消耗统计
- 快速入口：购买 / 详情

## 4. 筛选栏 (FilterBar)

### 发现页筛选栏

```
[全部] [微博] [知乎] [头条] [抖音] [B站] [36氪] [...更多▼]
[热度排序▼]  [今天▼]  [🔍 搜索关键词____________]
```

### 平台筛选

- Chip 样式，支持多选
- 选中状态：填充色 + 白色文字
- "更多" 展开剩余平台

### 排序选项

| 值 | 显示 |
|----|------|
| heat_desc | 热度最高 |
| heat_asc | 热度最低 |
| time_desc | 最新上榜 |
| cross_platform | 跨平台最多 |

### 时间范围

| 值 | 显示 |
|----|------|
| today | 今天 |
| 3d | 近3天 |
| 7d | 近7天 |

### 交互细节

- 筛选变更 → URL 参数同步（支持浏览器后退）
- 防抖搜索（300ms）
- 结果为空时显示空状态插图

## 5. 微信预览面板 (WeChatPreview)

### 外观

```
┌──────────────────┐
│ ┌──────────────┐ │  ← 手机壳（圆角 + 状态栏）
│ │ 🔋 9:41 AM   │ │
│ ├──────────────┤ │
│ │  文章标题     │ │
│ │  2026-04-04  │ │
│ │              │ │
│ │  正文内容...  │ │
│ │              │ │
│ │              │ │
│ │              │ │
│ └──────────────┘ │
│                  │
│ [切换模式▼]      │  ← 微信阅读 / 朋友圈卡片
└──────────────────┘
```

### 预览模式

| 模式 | 说明 |
|------|------|
| 微信阅读 | 模拟公众号文章阅读界面（默认） |
| 朋友圈卡片 | 模拟朋友圈分享卡片（封面+标题+摘要） |

### 同步逻辑

- 编辑器内容变更 → 300ms 防抖 → 重新渲染 HTML → 更新 iframe
- 样式模板切换 → 立即重新渲染
- 图片加载 → 占位符 → 实际图片

## 6. 内容列表 (ContentTable)

### Tab 切换

```
[全部(12)] [草稿(8)] [已发布(3)] [已归档(1)]
```

- 数字实时更新
- 切换 Tab 保持筛选条件

### 表格列

| 列 | 宽度 | 说明 |
|----|------|------|
| 选择框 | 40px | 批量操作用 |
| 标题 | 自适应 | 可点击跳转编辑 |
| Agent | 120px | 写作身份名称 |
| 状态 | 80px | 标签样式(草稿/已发布) |
| 创建时间 | 140px | 相对时间 |
| 操作 | 160px | 编辑/发布/删除 |

### 批量操作

选中 ≥1 项时，底部出现操作栏：

```
┌─────────────────────────────────────────────┐
│ 已选 3 项:  [批量发布]  [批量删除]  [取消选择] │
└─────────────────────────────────────────────┘
```
