> 最后更新：2026-04-04

## 流程概述

用户在创作页完成 AI 生成后，进入编辑和发布流程。

## 编辑能力

### 基础编辑

| 功能 | 说明 |
|------|------|
| 富文本编辑 | TipTap 编辑器，所见即所得 |
| 标题编辑 | 直接修改标题文字 |
| 摘要编辑 | 在标题下方编辑摘要 |
| 图片插入 | 本地上传 → COS → CDN URL |
| Markdown | 支持 Markdown 语法输入 |
| 撤销/重做 | Ctrl+Z / Ctrl+Shift+Z |

### AI 辅助编辑

**触发方式**：选中文本 → 出现浮动工具条

| 操作 | Credits | 说明 |
|------|---------|------|
| AI 改写 | 2 | 换一种说法，保持原意 |
| AI 扩写 | 3 | 基于选中内容展开 |
| AI 缩写 | 2 | 精简提炼 |

**流程**：
```
选中文本 → 浮动工具条 → 点击"AI改写"
  → 确认 Credits 消耗
  → 原文变灰，下方显示 AI 输出
  → 可选 [采纳] / [换一个] / [取消]
  → 采纳后替换原文
```

## 自动保存机制

```
编辑器内容变更
     │
     ├── 30秒无操作 → 自动保存
     ├── 切换页面前 → 自动保存
     └── 手动点击 → 立即保存

保存状态指示:
  ● 已保存 (灰色)
  ● 保存中... (蓝色旋转)
  ● 未保存 (黄色圆点)
```

## 微信预览同步

右栏微信手机壳预览与编辑器内容实时同步：

```
编辑器内容变更
     │
     ├── 防抖 300ms
     │
     ▼
渲染微信样式的 HTML
     │
     ▼
更新右栏 iframe
```

预览特性：
- 模拟微信公众号阅读界面
- 微信字体、行高、间距
- 图片自适应宽度
- 代码块样式
