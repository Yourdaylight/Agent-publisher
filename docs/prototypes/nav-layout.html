📐 原型说明：这是 Agent Publisher 的新导航布局原型。
核心变化：从 22 个页面收敛为 4 个核心 Tab（发现/创作/内容/数据） + 设置齿轮菜单。
交互：点击顶部 Tab 切换页面标识；点击 💎 展开 Credits 面板；点击 ⚙️ 展开设置菜单。

🔥 发现页 /discover

系统的默认首页和生产入口。聚合全网热榜，支持筛选、搜索、一键创作。

筛选栏

平台 Chip + 排序 + 时间 + 搜索

热点卡片列表

标题、热度条、跨平台标签、一键创作

快速创作弹窗

选 Agent + 模式 + 补充指令 → 生成

→ 详细原型见 discovery-page.html

✍️ 创作页 /create

三列布局写作工作台：左栏配置、中栏编辑、右栏微信预览。

左栏：写作配置

Agent 选择、风格、模板、素材、最近草稿

中栏：编辑器

AI 生成区 → TipTap 编辑器

右栏：微信预览

手机壳模拟 + 实时同步

→ 详细原型见 editor-page.html

📦 内容页 /content

草稿箱 + 已发布内容管理，支持批量操作。

Tab 切换

内容列表

标题、Agent、状态、操作

批量操作

批量发布、批量删除

Credits 使用情况 + 内容效果追踪。

Credits 概览

余额、消耗、生成次数统计

消耗趋势

消耗记录

→ 详细原型见 credits-dashboard.html
